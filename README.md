## 📌 Overview

This repository contains the **source code for my personal developer portfolio**.  
The portfolio presents my **skills, projects, education, experience, and resume** in a clean, modern, and professional layout.

The goal of this portfolio is simple:
- Clear communication
- Strong first impression
- Recruiter-friendly design
- Smooth but subtle animations

---

## 🧱 Tech Stack

- **Next.js 13**
- **React 18**
- **TypeScript**
- **Bootstrap / Reactstrap**
- **Lottie Animations**
- **Vercel (Deployment)**

---

## ✨ Key Sections

- **Hero / Greeting section** with animated background  
- **Skills section** with visual emphasis  
- **Projects section** showcasing selected work  
- **Education & Experience** timeline  
- **GitHub profile integration**  
- **Resume download** (`/resume.pdf`)
